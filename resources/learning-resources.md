# Learning Resources

A curated collection of resources to support skill development across all levels.

## Books

### Programming Fundamentals
- [Code Complete](https://www.amazon.com/Code-Complete-Practical-Handbook-Construction/dp/0735619670) - Practical handbook for writing quality code
- [The Pragmatic Programmer](https://pragprog.com/titles/tpp20/the-pragmatic-programmer-20th-anniversary-edition/) - Timeless advice for developers
- [Clean Code](https://www.amazon.com/Clean-Code-Handbook-Software-Craftsmanship/dp/0132350882) - Writing readable and maintainable code

### System Design & Architecture
- [Designing Data-Intensive Applications](https://dataintensive.net/) - Understanding distributed systems
- [System Design Interview](https://www.amazon.com/System-Design-Interview-insiders-Second/dp/B08CMF4YKW) - Practical guide for system design interviews
- [Domain-Driven Design](https://www.amazon.com/Domain-Driven-Design-Tackling-Complexity-Software/dp/0321125215) - Aligning software architecture with business needs

### Leadership & Soft Skills
- [The Mythical Man-Month](https://www.amazon.com/Mythical-Man-Month-Software-Engineering-Anniversary/dp/0201835959) - Classic book on software engineering challenges
- [Team Topologies](https://www.teamtopologies.com/) - Effective software team organization
- [Accelerate](https://www.amazon.com/Accelerate-Software-Performing-Technology-Organizations/dp/1942788339) - Key metrics for high-performing teams
