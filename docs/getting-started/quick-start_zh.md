# Vibe Coding 快速入门指南

几分钟内开始使用 Vibe Coding。

## 🌐 语言
- [English](quick-start.md) | [中文](quick-start_zh.md)

## 概述

本指南将帮助您快速了解如何采用 Vibe Coding 原则，并开始将快乐、创意的实践融入您的开发工作流程。

## 步骤 1：了解 Vibe Coding 哲学

Vibe Coding 强调：
- **创意表达**: 使用代码作为创造力和艺术性的画布
- **协作之乐**: 在共享的编码体验中找到快乐
- **正念实践**: 在编码时保持临在感和意图性
- **游戏化实验**: 在没有失败恐惧的情况下尝试新方法
- **社区连接**: 建立关系并培养积极的编码文化

## 步骤 2：选择您的起始级别

### 初学者 Vibe 编码者
- **最适合**: 对快乐编码感兴趣的开发新手
- **重点**: 正念实践、创意探索、积极协作
- **目录**: `skill-lists/beginner/`
- **在此开始，如果**: 您是编码新手或想为您的实践增添快乐

### 中级 Vibe 编码者
- **最适合**: 扩展创意和协作实践的开发人员
- **重点**: 协作之乐、创意表达、社区参与
- **目录**: `skill-lists/intermediate/`
- **在此开始，如果**: 您有一些经验并想增强团队合作

### 高级 Vibe 编码者
- **最适合**: 创建积极环境的有经验开发人员
- **重点**: 领导力、创意解决方案、社区建设
- **目录**: `skill-lists/advanced/`
- **在此开始，如果**: 您领导团队或项目并想转型文化

### 专家级 Vibe 编码者
- **最适合**: 塑造组织实践的高级开发人员
- **重点**: 文化转型、思想领导力、大规模社区建设
- **目录**: `skill-lists/expert/`
- **在此开始，如果**: 您影响组织政策和文化

## 步骤 3：访问 Vibe Coding 材料

### 选项 A：在线浏览
访问 Vibe Coding 仓库：
```
https://github.com/standup-coder/vibe-coding-skills
```

根据您的级别导航到适当的目录：
- `/skill-lists/beginner/` - 适用于新的 Vibe 编码者
- `/skill-lists/intermediate/` - 适用于成长中的 Vibe 编码者
- `/skill-lists/advanced/` - 适用于有经验的 Vibe 编码者
- `/skill-lists/expert/` - 适用于 Vibe 编码领导者

### 选项 B：本地下载
```bash
# 克隆仓库
git clone https://github.com/standup-coder/vibe-coding-skills.git
cd vibe-coding-skills

# 导航到您的级别
cd skill-lists/beginner/  # 或 intermediate/advanced/expert/
```

## 步骤 4：使用 Vibe Coding 清单

每个清单遵循以下格式：

```markdown
# 类别标题

- [ ] Vibe 技能名称 - 技能描述
- [ ] 另一个 Vibe 技能 - 另一个描述
- [x] 已完成 Vibe 技能 - 此项被勾选表示已完成
```

### 跟踪您的 Vibe 进度

1. **复制清单**: 复制一份以跟踪您的 Vibe 发展
   ```bash
   cp skill-lists/beginner/skills.md my_vibe_progress_tracker.md
   ```

2. **标记已完成的 Vibe**: 将您已实践的技能的 `[ ]` 替换为 `[x]`
   ```markdown
   # 之前
   - [ ] 编码时的意识呼吸

   # 之后
   - [x] 编码时的意识呼吸
   ```

3. **定期 Vibe 检查**: 定期审查您的进度并更新您的跟踪器

## 步骤 5：查找以 Vibe 为中心的活动

每个列表以以 Vibe 为中心的活动结束：

```markdown
## 活动与练习

1. [活动名称](https://example.com) - Vibe 建设活动的描述
2. [另一个活动](https://example.com) - 另一个活动的描述
```

使用这些活动来发展您的 Vibe Coding 技能。

## 步骤 6：设定 Vibe 目标和里程碑

### 进度跟踪公式
```
Vibe 掌握率 = (已实践的 Vibe 技能 / 总 Vibe 技能) × 100%
```

### 里程碑目标
- **25%**: 良好的 Vibe 基础，继续流动
- **50%**: 中途 Vibe 里程碑，保持律动
- **75%**: 接近 Vibe 完成，快到了
- **100%**: Vibe 成就达成，进阶到下一级别

## 入门后的下一步

1. **进阶到下一个 Vibe 级别**: 逐渐进阶到更高级别
2. **分享您的 Vibe**: 将正能量传播给他人
3. **为社区做贡献**: 为 Vibe Coding 项目增添内容
4. **探索创意编码**: 更深入地研究艺术编码
