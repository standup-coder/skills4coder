# 初学者开发者技能清单

新软件开发者开始其旅程的基本技能。

## 🌐 语言
- [English](beginner.md) | [中文](beginner_zh.md)

## 编程基础

- [ ] **基本语法和语义** - 了解变量、常量、运算符和基本语言结构
- [ ] **数据类型和变量** - 声明和使用原始数据类型（数字、字符串、布尔值）
- [ ] **控制结构** - 使用if/else、switch/case和三元运算符进行条件逻辑
- [ ] **循环** - 实现for、while和do-while循环以重复操作
- [ ] **函数和方法** - 创建带参数和返回值的可重用代码块
- [ ] **基本数据结构** - 使用数组/列表和字典/映射存储数据集合
- [ ] **输入/输出操作** - 从控制台/文件读取和写入
- [ ] **异常处理** - 使用try/catch块优雅地管理错误

## 开发环境

- [ ] **文本编辑器/IDE设置** - 配置和使用代码编辑器（VSCode、IntelliJ等）
- [ ] **命令行界面** - 在终端/命令提示符中导航和执行命令
- [ ] **文件系统操作** - 创建、移动、复制和组织文件/目录
- [ ] **包管理器** - 使用npm、pip、Maven等安装和管理依赖项
- [ ] **基本调试** - 使用调试器工具识别和修复代码中的问题
- [ ] **环境变量** - 了解和配置环境变量

## 版本控制

- [ ] **Git基础** - 了解git init、git clone、git status命令
- [ ] **文件跟踪** - 使用git add、git rm、.gitignore添加、删除和忽略文件
- [ ] **提交更改** - 使用git commit创建有意义的提交
- [ ] **查看历史** - 使用git log、git diff和git show检查更改
- [ ] **分支基础** - 使用git branch创建、切换和删除分支
- [ ] **远程仓库** - 将本地仓库连接到远程主机（GitHub、GitLab）

## 问题解决和算法

- [ ] **算法思维** - 将复杂问题分解为更小、可管理的任务
- [ ] **基本数学运算** - 了解算术、比较和逻辑运算
- [ ] **字符串操作** - 使用连接、分割和搜索处理文本数据
- [ ] **数组/列表操作** - 访问、修改和遍历集合数据
- [ ] **搜索算法** - 实现线性搜索以查找元素
- [ ] **排序算法** - 了解冒泡排序和选择排序概念
- [ ] **调试逻辑** - 在简单程序中识别和修复逻辑错误

## 软件开发概念

- [ ] **代码可读性** - 编写整洁、适当缩进、适当间距的代码
- [ ] **命名约定** - 为变量、函数和类使用描述性名称
- [ ] **注释和文档** - 编写清晰的注释以解释代码功能
- [ ] **代码组织** - 将代码结构化为逻辑部分和文件
- [ ] **测试基础** - 编写简单测试以验证代码功能
- [ ] **重构原则** - 了解改进现有代码的重要性

## 基本网络技术（适用于网络开发者）

- [ ] **HTML基础** - 使用语义HTML元素创建正确的文档结构
- [ ] **CSS基础** - 使用选择器、颜色、字体和基本布局设计网页
- [ ] **JavaScript介绍** - 使用基本JS为网页添加交互性
- [ ] **DOM操作** - 使用JavaScript与HTML元素交互
- [ ] **浏览器开发者工具** - 使用浏览器工具进行调试和检查

## 专业技能

- [ ] **技术沟通** - 用简单术语解释技术概念
- [ ] **自学** - 查找和利用在线资源学习新概念
- [ ] **时间管理** - 将项目分解为具有现实时间表的可管理任务
- [ ] **注意细节** - 仔细检查代码的语法和逻辑错误
- [ ] **坚持** - 尽管最初受挫也要继续解决问题

## 学习资源

1. **在线课程**: freeCodeCamp、Codecademy、Coursera编程入门课程
2. **书籍**: 《JavaScript精粹》、《用Python自动做一些无聊的事》、《Head First编程》
3. **文档**: 您选择的编程语言的官方文档
4. **练习平台**: LeetCode、HackerRank、Codewars用于算法练习
5. **社区**: Stack Overflow、Reddit编程社区、Discord服务器
