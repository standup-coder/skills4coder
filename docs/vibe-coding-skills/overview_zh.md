# Vibe Coding 技能概览

本项目中可用的 Vibe Coding 技能的综合概览。

## 🌐 语言
- [English](overview.md) | [中文](overview_zh.md)

## 介绍

Vibe Coding 代表了软件开发的独特方法，强调在编码过程中融入创造力、协作、正念和乐趣。该项目提供按经验级别组织的结构化技能清单，专注于超越纯技术技能的编码特殊方面。

## 什么是 Vibe Coding？

Vibe Coding 是一种编程方法，强调：

- **创意表达**: 使用代码作为艺术表达和实验的画布
- **协作之乐**: 在共享编码体验中找到快乐和满足感
- **正念实践**: 在编写代码时保持临在感和意图性
- **游戏化实验**: 以好奇心和无失败恐惧的方法处理编码挑战
- **社区连接**: 建立关系并培育积极的编码文化

## Vibe Coding 技能结构

### 共同元素
所有 Vibe Coding 技能清单共享相同结构：

```markdown
# 类别标题

- [ ] Vibe 技能名称 - 技能描述
- [ ] 另一个 Vibe 技能 - 另一个描述
- [x] 已完成 Vibe 技能 - 表示已完成

## 活动与练习

1. [活动名称](https://example.com) - 描述
2. [另一个活动](https://example.com) - 描述
```

### 每个清单内的类别
每个技能清单分为以 Vibe 为中心的类别：
- **创意表达**: 将代码用作创意媒介的技能
- **协作实践**: 与他人快乐编码的技能
- **正念技巧**: 有意图和正念编码的技能
- **游戏化学习**: 以快乐方式实验和学习的技能
- **社区建设**: 培育积极编码环境的技能

## 如何使用 Vibe Coding 技能清单

### 自我评估
1. 确定您当前的 Vibe Coding 经验级别
2. 选择适当的技能清单
3. 遍历每个技能并评估您的 Vibe 相关知识
4. 将您已实践的技能标记为已完成 ([x])
5. 专注于未标记的技能

### 快乐学习路径创建
1. 查看您目标级别的技能清单
2. 从较低级别识别先决 Vibe 技能
3. 创建以积极体验为重点的定制学习序列
4. 使用提供的活动来发展每项技能
5. 定期练习并庆祝进步

### 团队发展
1. 将技能清单用作快乐开发实践的框架
2. 识别团队 Vibe 优势和成长机会
3. 创建以积极为重点的个性化发展计划
4. 跟踪团队在创造积极编码环境方面的进度
5. 计划协作学习活动
