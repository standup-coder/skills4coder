## 关于 Vibe Coding

Vibe Coding 是一种编程方法，强调：

- **创意表达**: 使用代码作为创造力和艺术性的画布
- **协作之乐**: 在共享编码体验中找到快乐
- **正念实践**: 在编码时保持临在感和意图性
- **游戏化实验**: 在没有失败恐惧的情况下尝试新方法
- **社区连接**: 建立关系并培养积极的编码文化

技能清单按经验级别组织：

- **初学者**: 适用于开始 Vibe Coding 之旅的人
- **中级**: 适用于在基础技能之上构建的 Vibe 编码者
- **高级**: 适用于处理复杂协作挑战的经验丰富的 Vibe 编码者
- **专家**: 适用于创建积极环境的 Vibe Coding 领导者

每个级别都包含与该阶段以 Vibe 为中心的开发相关的特定技能。
