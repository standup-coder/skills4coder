# 初学者 Vibe Coding 技能清单

这份清单涵盖了对快乐、创意编码体验感兴趣的开发人员的基础技能。

## 🌐 语言
- [English](skills_en.md) | [中文](skills_zh.md) | [综合版](skills.md)

## 正念编码实践

- [ ] 编码时的意识呼吸 - 在深入复杂代码之前进行正念呼吸
- [ ] 设定积极意向 - 以积极的目标和态度开始编码会话
- [ ] 定期休息 - 实施短暂休息以保持专注和快乐
- [ ] 身体意识 - 在编码时保持对身体舒适的意识
- [ ] 无评判意识 - 在观察代码时避免严厉的自我批评

## 创意编码探索

- [ ] 尝试视觉代码艺术 - 创建代码的美学排列
- [ ] 多彩注释艺术 - 创意使用注释以增加视觉趣味
- [ ] 创意命名约定 - 在表达性的变量和函数名中找到快乐
- [ ] 代码中的表情符号使用 - 使用表情符号为注释和日志增添个性
- [ ] 游戏化控制台消息 - 在控制台输出中添加令人愉悦的消息

## 快乐学习方法

- [ ] 庆祝小胜利 - 认可在编码中的小成就
- [ ] 重新框定错误为学习 - 将错误视为成长的机会
- [ ] 好奇心驱动的探索 - 对代码提出"如果"问题
- [ ] 通过游戏学习 - 在没有特定目标的情况下实验代码
- [ ] 积极肯定 - 在编码挑战中使用积极的自言自语

## 积极协作基础

- [ ] 友好沟通 - 在代码注释和文档中使用鼓励性语言
- [ ] 自信提问 - 毫不羞耻或犹豫地寻求帮助
- [ ] 清晰表达想法 - 尊重地传达想法和建议
- [ ] 积极倾听 - 关注他人的编码见解
- [ ] 分享学习时刻 - 讨论发现和突破

## 在编码中建立信心

- [ ] 设定可管理的目标 - 将复杂任务分解为可实现的步骤
- [ ] 积极跟踪进度 - 专注于改进而不是差距
- [ ] 积极的自我对话 - 在编码时使用支持性的内心语言
- [ ] 认识成长 - 识别编码能力的个人发展
- [ ] 在过程中找到快乐 - 享受旅程，而不仅仅是目的地
