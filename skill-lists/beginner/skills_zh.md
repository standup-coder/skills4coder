# 初学者技能清单

这份清单涵盖了编程初学者应该掌握的基础技能。

## 🌐 语言
- [English](skills_en.md) | [中文](skills_zh.md) | [综合版](skills.md)

## 编程基础

- [ ] 变量与数据类型 - 了解如何声明变量和使用不同的数据类型
- [ ] 条件语句 - 使用 if/else 语句在代码中做决策
- [ ] 循环 - 使用 for/while 循环重复代码块
- [ ] 函数 - 使用参数和返回值创建可重用的代码块
- [ ] 数组与列表 - 存储和操作数据集合
- [ ] 错误处理 - 优雅地管理异常和错误

## 版本控制

- [ ] Git 基础命令 - 了解 git init, git add, git commit, git status
- [ ] 创建仓库 - 设置本地和远程仓库
- [ ] 提交更改 - 使用描述性提交消息保存更改
- [ ] 分支管理 - 在不同的开发分支之间创建和切换
- [ ] 合并 - 合并来自不同分支的更改

## 开发环境

- [ ] 代码编辑器设置 - 高效地配置和使用代码编辑器
- [ ] 终端使用 - 在终端/Shell 中导航和执行命令
- [ ] 文件系统操作 - 从命令行管理文件和目录
- [ ] 包管理器使用 - 使用 npm、pip 等工具安装和管理依赖项

## 基本算法和数据结构

- [ ] 数组操作 - 常见操作如搜索、排序和操作
- [ ] 链表概念 - 了解基本链表结构和用法
- [ ] 栈和队列 - 了解 LIFO 和 FIFO 数据结构
- [ ] 基本排序算法 - 冒泡排序、选择排序、插入排序
- [ ] 基本搜索算法 - 线性搜索、二分搜索

## 测试基础

- [ ] 单元测试概念 - 了解单元测试的目的
- [ ] 断言 - 使用断言验证代码行为
- [ ] 测试驱动开发 - 在实现功能之前编写测试

## Web 开发基础

- [ ] HTML 结构 - 使用语义化 HTML 创建适当的文档结构
- [ ] CSS 布局 - 在网页上设置元素样式和定位
- [ ] JavaScript DOM 操作 - 动态地与网页元素交互
- [ ] HTTP 请求基础 - 向 Web 服务器发出 GET/POST 请求

## 数据库基础

- [ ] 关系型数据库概念 - 了解表、行、列、关系
- [ ] SQL 查询 - 编写基本的 SELECT、INSERT、UPDATE、DELETE 语句
- [ ] 基本表设计 - 使用适当的数据类型和约束创建表

## 学习资源

1. [freeCodeCamp](https://www.freecodecamp.org/) - 涵盖 Web 开发基础的免费编码课程
2. [MDN Web Docs](https://developer.mozilla.org/) - 全面的 Web 技术文档
3. [The Odin Project](https://www.theodinproject.com/) - 全栈 Web 开发课程
4. [Khan Academy](https://www.khanacademy.org/) - 计算机科学和编程课程
5. [Coursera](https://www.coursera.org/) - 大学级别的编程课程
