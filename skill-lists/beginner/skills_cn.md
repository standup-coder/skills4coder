# 初学者技能清单 (Beginner Level)

这份清单涵盖了编程初学者应该掌握的基础技能。

## 编程基础 (Programming Fundamentals)

- [ ] 变量与数据类型 (Variables and Data Types)
- [ ] 条件语句 (Conditional Statements)
- [ ] 循环 (Loops)
- [ ] 函数 (Functions)
- [ ] 数组与列表 (Arrays and Lists)
- [ ] 错误处理 (Error Handling)

## 版本控制 (Version Control)

- [ ] Git 基础命令 (Basic Git Commands)
- [ ] 创建仓库 (Creating Repositories)
- [ ] 提交更改 (Committing Changes)
- [ ] 分支管理 (Branching)
- [ ] 合并代码 (Merging)

## 开发环境 (Development Environment)

- [ ] 代码编辑器设置 (Setting up Code Editor)
- [ ] 终端使用 (Using Terminal)
- [ ] 文件系统操作 (File System Operations)
- [ ] 包管理器使用 (Package Manager Usage)

## 算法与数据结构基础 (Basic Algorithms and Data Structures)

- [ ] 数组操作 (Array Operations)
- [ ] 链表概念 (Linked List Concepts)
- [ ] 栈与队列 (Stacks and Queues)
- [ ] 排序算法基础 (Basic Sorting Algorithms)
- [ ] 搜索算法基础 (Basic Search Algorithms)

## 测试基础 (Testing Fundamentals)

- [ ] 单元测试概念 (Unit Testing Concepts)
- [ ] 断言 (Assertions)
- [ ] 测试驱动开发 (Test Driven Development)

## Web 开发基础 (Web Development Basics)

- [ ] HTML 结构 (HTML Structure)
- [ ] CSS 布局 (CSS Layout)
- [ ] JavaScript DOM 操作 (JavaScript DOM Manipulation)
- [ ] HTTP 请求基础 (HTTP Requests Basics)

## 数据库基础 (Database Basics)

- [ ] 关系型数据库概念 (Relational Database Concepts)
- [ ] SQL 查询 (SQL Queries)
- [ ] 表设计基础 (Basic Table Design)
