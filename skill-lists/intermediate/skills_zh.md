# 中级 Vibe Coding 技能清单

这份清单涵盖了扩展其创意和协作编码实践的开发人员的技能。

## 🌐 语言
- [English](skills_en.md) | [中文](skills_zh.md) | [综合版](skills.md)

## 协作快乐技巧

- [ ] 促进快乐的结对编程 - 在配对会话中创造积极的环境
- [ ] 包容性结团编程 - 确保每个人在小组编码中平等参与
- [ ] 积极冲突解决 - 以同理心和尊重处理分歧
- [ ] 庆祝团队胜利 - 认可在项目中的集体成就
- [ ] 鼓励冒险 - 为实验和错误创造安全空间

## 代码中的创意表达

- [ ] 美学代码格式化 - 以视觉吸引力为前提组织代码
- [ ] 主题代码项目 - 围绕有趣的概念或主题开发项目
- [ ] 创意重构 - 在保持或增强优雅性的同时改进代码
- [ ] 表达性注释 - 编写不仅传达信息而且传达个性的注释
- [ ] 代码诗歌 - 创作读起来优美且完美运行的代码

## 正念开发实践

- [ ] 有意识的代码审查 - 提供善意和学习导向的反馈
- [ ] 冥想整合 - 将简短的正念练习纳入编码常规
- [ ] 无压力调试 - 以平静的好奇心而非挫败感处理错误
- [ ] 心流状态培养 - 创造最佳专注和生产力的条件
- [ ] 边界设定 - 在编码项目中保持健康的工作生活平衡

## 社区参与技能

- [ ] 欢迎新编码者 - 让新人感到被包含和珍视
- [ ] 同理心知识分享 - 以耐心和理解教导他人
- [ ] 积极指导 - 以鼓励和支持引导他人
- [ ] 建设性反馈文化 - 培育欢迎反馈的环境
- [ ] 包容性工作坊主持 - 为多样化受众领导教育会话

## 积极的教学和指导

- [ ] 耐心解释技巧 - 清晰地分解复杂概念
- [ ] 鼓励成长心态 - 帮助他人将挑战视为学习机会
- [ ] 庆祝他人的成功 - 热情地支持队友的成就
- [ ] 建模积极实践 - 在编码中展示快乐和正念
- [ ] 创造学习机会 - 识别他人成长的时刻
